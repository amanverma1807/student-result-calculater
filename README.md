# Student Result Generator

A Python automation project built using openpyxl.

## Features
- Reads student marks from Excel
- Calculates total marks
- Calculates percentage
- Assigns grades
- Finds topper
- Generates styled Excel report
- Pass/Fail status
- Colored formatting

## Technologies Used
- Python
- openpyxl
- os

## Output
The program generates a final Excel report automatically inside the output folder.
